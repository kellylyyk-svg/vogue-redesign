# Vogue Video Curation Page

## Overview
A premium, dark-mode video curation page styled after Vogue Korea's high-fashion aesthetic. 
Features infinite scrolling masonry grids, concave layout masks, and specific interactions.

## How to Run
Simply open `index.html` in any modern web browser.

## Customization
- **Videos**: Uncomment the `<video>` tag logic in `index.html` and `main.js` to replace the placeholder images.
- **Images**: Images are currently hosted on Google Cloud Storage. Replace `src` attributes in `index.html` and `main.js` with your own assets.
